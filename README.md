# TeroViwer
Program do przeglądania segmentów map pobranych z Geoportal. 
Czasami zdarza się, że pobrane segmenty nie można po prostu skleić w jedną mapę bitową, ponieważ jej rozmiar może przekroczyć 30000 pikseli. 
Pozostawienie segmentów i łączenie ich kilku do podglądu jest lepszym rozwiązaniem.

Nie udostępniam kodu.
